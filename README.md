# Pro_module
项目模块
### 任务
- 项目类型为 Web 项目
- 需要额外添加模块，至少有 dao、service、entity 模块
- 为各个模块设置好依赖关系
- 上传项目到自己的 github

## 模块化的优点
- 1、非常便于内容的组织和管理；
- 2、适用于响应式框架；
- 3、融合卡片和极简风格；
- 4、基于网格，能在任何项目平台上使用；
